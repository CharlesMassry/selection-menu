SelectionMenu

SelectionMenu is a small, self-contained, framework-agnostic JavaScript module that displays a custom context menu when the user selects text on the page.

This menu may offer a search feature, dictionary lookup, post to Facebook or similar.

Select some text with the mouse or by tapping!

Now try selecting the other way (e.g. right to left)!

Notice how the popover menu conveniently appears where you finished the selection.

Only selections at least 5 characters long trigger the menu. Try selecting this word.
